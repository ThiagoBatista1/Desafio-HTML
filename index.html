Introdução

Retornar ao Inicio

No contexto da web, o cliente (client) é o lado que consome informações, geralmente representado pelo navegador web que utilizamos para acessar os sites. Por outro lado, o servidor (server) é o lado que serve essas informações para o cliente, armazenando e processando os dados solicitados. A comunicação entre eles é feita através da internet, o cliente (client) manda uma requisição http (hypertext transfer protocol) e o servidor (server) retorna o HTML, CSS, Javascript, e etc. É importante entender essa dinâmica para compreender como as páginas da web são carregadas e exibidas para os usuários. Uma das ferramentas usadas no universo web é o HTML (hypertext markup language), que não é uma linguagem de programação, mas sim uma linguagem de marcação que utiliza textos delimitados (tags) para criar documentos estruturados que os navegadores podem interpretar e exibir aos usuários. Cada tag tem uma função e comportamento específico, permitindo a criação de diferentes elementos em uma página web.

Ferramentas para Facilitar o Desenvolvimento

Retornar ao Inicio

Ao iniciar o desenvolvimento de uma página web, é essencial contar com ferramentas que facilitem o processo. A inspeção de elementos (tecla F12 no Windows) permite examinar e depurar o código em tempo real. A extensão Live Server no Visual Studio Code oferece a conveniência de visualizar as alterações no HTML em tempo real. O uso do padrão "Index" para nomear a página inicial do projeto simplifica a navegação. Essas ferramentas facilitam o desenvolvimento, proporcionando um ambiente mais produtivo.

Conceito de Tag e Atributo

Retornar ao Inicio

As tags são elementos fundamentais no HTML, formados por um sinal de menor e um de maior, com uma palavra entre eles representando o nome da tag. O HTML é dividido em três partes principais: html, head e body , sendo o body responsável por tudo o que é exibido para o usuário. As tags podem ter atributos, esses atributos podem ser divididos como genéricos ou específicos, os genéricos são atributos que toda tag pode ter, como por exemplo id, style e class, que permitem definir propriedades específicas ou identificadores para os elementos. Já os atributos específicos, são atributos que tags em específico pode ter, como por exemplo type, src e width, que têm diferentes finalidades e funcionalidades.

Construção do Texto

Retornar ao Inicio

Para formatar o texto em uma página HTML, utilizamos uma variedade de tags, como h1 a h6 para títulos de diferentes níveis de importância, p para parágrafos, blockquote para citações, u para sublinhar, mark para destacar com uma cor de fundo, sup para itens elevados, hr para criar uma linha horizontal e small para texto em tamanho reduzido. É possível encontrar uma referência completa dessas e outras tags no site W3schools , que é uma fonte valiosa de informações para desenvolvedores web.
