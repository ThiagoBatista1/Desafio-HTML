  1. Introdução

    Retornar ao Inicio

    No contexto da web, o cliente (client) é o lado que consome informações, geralmente representado pelo navegador web que utilizamos para acessar os sites. Por outro lado, o servidor (server) é o lado que serve essas informações para o cliente, armazenando e processando os dados solicitados. A comunicação entre eles é feita através da internet, o cliente (client) manda uma requisição HTTP (Hypertext Transfer Protocol) e o servidor (server) retorna o HTML (Hypertext Markup Language), CSS (Cascading Style Sheets) , Javascript, e etc. É importante entender essa dinâmica para compreender como as páginas da web são carregadas e exibidas para os usuários. Uma das ferramentas usadas no universo web é o HTML, que não é uma linguagem de programação, mas sim uma linguagem de marcação que utiliza textos delimitados (tags) para criar documentos estruturados que os navegadores podem interpretar e exibir aos usuários. Cada tag tem uma função e comportamento específico, permitindo a criação de diferentes elementos em uma página web.

  2. Ferramentas para Facilitar o Desenvolvimento

    Retornar ao Inicio

    Ao iniciar o desenvolvimento de uma página web, é essencial contar com ferramentas que facilitem o processo. A inspeção de elementos (tecla F12 no Windows) permite examinar e depurar o código em tempo real. A extensão Live Server no Visual Studio Code oferece a conveniência de visualizar as alterações no HTML (Hypertext Markup Language) em tempo real. O uso do padrão "Index" para nomear a página inicial do projeto simplifica a navegação. Essas ferramentas facilitam o desenvolvimento, proporcionando um ambiente mais produtivo.

  3. Conceito de Tag e Atributo

    Retornar ao Inicio

    As tags são elementos fundamentais no HTML (Hypertext Markup Language), formados por um sinal de menor e um de maior, com uma palavra entre eles representando o nome da tag. O HTML é dividido em três partes principais: html, head e body , sendo o body responsável por tudo o que é exibido para o usuário. As tags podem ter atributos, esses atributos podem ser divididos como genéricos ou específicos, os genéricos são atributos que toda tag pode ter, como por exemplo id , style e class, que permitem definir propriedades específicas ou identificadores para os elementos. Já os atributos específicos, são atributos que tags em específico pode ter, como por exemplo type , src e width, que têm diferentes finalidades e funcionalidades.

  4. Construção do Texto

    Retornar ao Inicio

    Para formatar o texto em uma página HTML (Hypertext Markup Language), utilizamos uma variedade de tags, como h1 a h6 para títulos de diferentes níveis de importância, p para parágrafos, blockquote para citações, u para sublinhar, mark para destacar com uma cor de fundo, sup para itens elevados, hr para criar uma linha horizontal e small para texto em tamanho reduzido. É possível encontrar uma referência completa dessas e outras tags no site W3schools , que é uma fonte valiosa de informações para desenvolvedores web.

  5. Listas

    Retornar ao Inicio

    As listas em HTML (Hypertext Markup Language) podem ser ordenadas (ol) ou não ordenadas (ul) , e ambas podem conter itens representados pela tag li . Nas listas ordenadas, os itens são numerados automaticamente de acordo com a ordem em que são inseridos, enquanto nas não ordenadas, são utilizados marcadores para cada item. É possível criar listas aninhadas, ou seja, listas dentro de itens de outras listas, para organizar e estruturar o conteúdo de forma hierárquica.

  6. Links

    Retornar ao Inicio

    Os links em HTML (Hypertext Markup Language) são criados com a tag a, que possui o atributo href para especificar o destino do link, como uma URL (Uniform Resource Locator) externa ou uma página interna do mesmo site. O atributo target determina se o link será aberto na mesma guia (_self) ou em uma nova guia (_blank). Além disso, o atributo title pode ser utilizado para exibir uma mensagem ao passar o mouse sobre o link. Também é possível ancorar links para partes específicas de uma página utilizando IDs , o que permite aos usuários navegar facilmente dentro do mesmo documento HTML.
